Price Pattern Analysis

The program loops through a list of prices and performs several checks. As it reads each price, it keeps track of how many items cost more than $5 and how many cost less than $3 by increasing counters (overFive and underThree). At the same time, it updates two variables—highest and lowest—to store the maximum and minimum prices it finds in the list.

The code also checks whether the prices are in ascending order. It does this by comparing each price to the one before it; if any price is smaller than the previous one, the program sets ascending to false. After processing all prices, the program prints out the results: the number of items over $5, the number under $3, whether the list is in ascending order, and the highest and lowest prices using formatted output.
